A tooltip is a message box that is displayed when a user hovers over or gives focus to a UI element.

#### When to use

- Describe icon buttons
- When more information is useful in helping a user make decisions
- When an element needs more context or explanation

# Tooltip Coordinator

```typescript
import {TooltipCoordinator} from '@frontapp/ui-kit';
```

The `TooltipCoordinator` is a component that handles all the displaying logic and when to display a tooltip.
You can specify conditions for the tooltip to be rendered. The available options are:

- `overflow`: Controls if the tooltip should be enabled by checking if the text is overflowing. Only supports single line.
- `explicit`: Controls if the tooltip should be enabled via the `isEnabled` option.
