# Stories

## Min and Max Dates

<Canvas>
  <Story id="components-date-picker--min-max-date" />
</Canvas>

You can specify the min and max dates when constructing the component. This will limit the user to only select a certain
range of dates. In this example we only allow selecting a week before or after the current date.

<br />

## Different Start Date
