# Action Menu

<Canvas>
  <Story id="components-action-menu--basic" />

An action menu is a small menu that appears from a triple dots button. The button is not customizable, only the content
of the menu is customizable.

<br />

# Action Menu Item

The `ActionMenuItem` is the component that you should use when using the `ActionMenu`. If the label overflows it will automatically render
a tooltip for the overflowing content.
